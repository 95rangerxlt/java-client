Introduction

API Endpoint: https://api.blockcypher.com

Libraries

  • cURL
  • Java
  • Ruby
  • Python

Install

http://curl.haxx.se/

Add the following to your maven POM

<dependency>
  <groupId>com.blockcypher</groupId>
  <artifactId>java-client</artifactId>
  <version>1.0</version>
</dependency>

You will also find the JAVA API Docs and sources on http://blockypher.github.com/java-client


Get the gem

sudo gem install --source http://blockypher.github.com/ruby-client blockcypher

Or bundler

gem 'blockcypher', :git => 'http://blockypher.github.com/ruby-client'

Install with pip

sudo pip install --index-url http://blockypher.github.com/python-client --upgrade blockcypher

With easy_install

sudo easy_install --index-url http://blockypher.github.com/python-client --upgrade blockcypher

Transaction Service

With Transaction Service you can get existing Transaction details.

You can create a skeleton transaction, initiate it, sign the returned transaction and submit it.

Transaction transaction = context.getTransactionService()
